# MyEdu

#### 介绍
知识付费平台微信小程序的实现，基于uniapp

#### 软件架构
软件架构说明


#### 安装教程

1. 下载 [HBuilder](https://www.dcloud.io/hbuilderx.html) 开发工具
2. 下载代码包 https://ext.dcloud.net.cn/plugin?id=5551
3. 导入到HBuilder 开发工具中
